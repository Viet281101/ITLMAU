◆ vector_new()

vector_t * vector_new ( void  )

créé un vector en allouant (malloc) une structure vector de size 0 et max_size égal à 1024. Pour le champ data, max_size éléments (ici des int) sont alloués.

La fonction retourne le pointeur vers la nouvelle structure ainsi paramétrée. Ce pointeur de vector doit être libéré à l'aide de vector_delete.

le pointeur vers le nouveau vector alloué et initialisé.

ajoute un élément (le int data) à la fin du vector pointé par vec.

Si le vecteur n'a plus assez d'espace de stackage (size == max_size), max_size est doublée et data est réallouée en conséquence.

Paramètres

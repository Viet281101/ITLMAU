
# IA pour le jeu d'Othello

Un jeu d'Othello avec l'IA en utilse le langage C et openGL GLUT

## Compilation

### Prérequis :
  
    ```
    sudo apt-get install make
    ```
    et
    ```
    sudo apt-get install libglut3-dev
    ```
  
### Pour compiler le jeu :

    ```
    make
    ```

## Lancement :
    ```
    ./othello
    ```

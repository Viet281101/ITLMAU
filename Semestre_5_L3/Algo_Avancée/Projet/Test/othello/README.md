# Othello

## Compilation

```
$ make
```

## Usage

> Lance le jeu avec les N(oirs) et les B(blancs)
>
> Arguments possibles : `humain` ou `h` | `minimax` ou `m` | `alphabeta` ou `a`

```
$ ./othello <N> <B>
```

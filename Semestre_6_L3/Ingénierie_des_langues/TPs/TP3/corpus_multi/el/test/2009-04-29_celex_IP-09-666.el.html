IP/09/666 Βρυξέλλες 29 Απριλίου 2009Χρηματοπιστωτικές υπηρεσίες: Η Επιτροπή προτείνει μέτρα καλύτερης προστασίας των επενδυτών για τις δέσμες επενδυτικών προϊόντων για ιδιώτες επενδυτέςΗ Ευρωπαϊκή Επιτροπή έχει δεσμευθεί να επιφέρει σημαντικές βελτιώσεις στα μέτρα προστασίας των επενδυτών για τα κυριότερα επενδυτικά προϊόντα που αγοράζουν οι ιδιώτες επενδυτές. Οι ασυνέπειες των υφισταμένων προτύπων μπορούν να είναι καταστρεπτικές για τους επενδυτές και μπορούν να οδηγήσουν σε στρεβλώσεις του ανταγωνισμού στην αγορά επενδύσεων για μικροεπενδυτές. Τα συμπεράσματα της Επιτροπής, που παρουσιάζονται στην ανακοίνωση για τις δέσμες επενδυτικών προϊόντων για ιδιώτες επενδυτές, είναι ότι οι απαιτήσεις πληροφόρησης για τα προϊόντα και οι κανόνες για τις πωλήσεις προϊόντων πρέπει να βελτιωθούν και να αποκτήσουν μεγαλύτερη συνέπεια. Στην ανακοίνωση συνοψίζονται προτάσεις για μια νέα, οριζόντια νομοθετική προσέγγιση, που θα βασίζεται στα καλύτερα σημεία των υφισταμένων απαιτήσεων και θα τα εφαρμόζει σε όλα τα σχετικά προϊόντα. Η Επιτροπή θα αρχίσει εργασίες για τις λεπτομερείς νομοθετικές προτάσεις που απαιτούνται για αυτή την νέα προσέγγιση, και θα δώσει κατευθυντήριες γραμμές για τις εργασίες μέχρι το τέλος του 2009. Ο κ. Charlie McCreevy, Επίτροπος αρμόδιος για θέματα εσωτερικής αγοράς και υπηρεσιών, προέβη στην ακόλουθη δήλωση: «Η οικονομική κρίση έδειξε για μια ακόμη φορά πόσο σημαντικό είναι να είναι σε θέση οι μικροεπενδυτές να λαμβάνουν επενδυτικές αποφάσεις με πλήρη επίγνωση των σχετικών κινδύνων. Έχει καίρια σημασία να λαμβάνουν οι επενδυτές πληροφορίες τις οποίες μπορούν να εμπιστευθούν και να κατανοήσουν για τις επενδύσεις που πραγματοποιούν, και οι πωλητές των επενδύσεων να αντιμετωπίζουν πάντα τους επενδυτές δίκαια και να λαμβάνουν υπόψη τις ανάγκες τους. Το φιλόδοξο πρόγραμμα εργασιών που ανακοινώσαμε σήμερα είναι μελετημένο ώστε να εξασφαλίσει σημαντική πρόοδο προς αυτό το στόχο, εξασφαλίζοντας την κανονιστική ρύθμιση της πληροφόρησης και των πωλήσεων των δεσμών επενδυτικών προϊόντων για ιδιώτες επενδυτές με βάση υψηλά και συνεπή πρότυπα.» Οι προτάσεις εστιάζουν στην πληροφόρηση για τα προϊόντα και τις διαδικασίες πώλησης για όλες τις δέσμες επενδυτικών προϊόντων για ιδιώτες επενδυτές, όπως είναι τα επενδυτικά κεφάλαια, οι επενδύσεις οι βασισμένες στις ασφαλίσεις και οι διάφοροι τύποι δομημένων προϊόντων. Τα προϊόντα αυτά δεσπόζουν στην αγορά επενδύσεων για μικροεπενδυτές, ικανοποιούν παρεμφερείς ανάγκες των επενδυτών και θέτουν συγκρίσιμα και σημαντικά προβλήματα όσον αφορά την προστασία των επενδυτών. Η ανακοίνωση δεν περιέχει λεπτομερείς νομοθετικές προτάσεις στην παρούσα φάση, αλλά ορίζει τις υψηλού επιπέδου αρχές μιας οριζόντιας προσέγγισης σχετικά με την πληροφόρηση για τα προϊόντα και τις πρακτικές πώλησης, δεσμεύει δε την Επιτροπή να υποβάλει νομοθετικές προτάσεις που θα οδηγήσουν σε αυτά τα αποτελέσματα. ΙστορικόΗ αγορά για δέσμες επενδυτικών προϊόντων για ιδιώτες επενδυτές είναι πολύ μεγάλη, και εκτιμάται ότι η αξία της μπορεί να ήταν έως και 8 τρισ. ευρώ στο τέλος του 2008. Τα προϊόντα αυτά μπορούν να προσφέρουν σημαντικά οφέλη στους μικροεπενδυτές. Ωστόσο, συχνά είναι περίπλοκα και είναι δύσκολο για τους επενδυτές να τα κατανοούν πλήρως, ιδίως όσον αφορά τους κινδύνους και τις δαπάνες τους. Οι παρούσες προτάσεις είναι μέρος του προγράμματος μεταρρύθμισης που παρουσιάζεται στην ανακοίνωση της Επιτροπής για το εαρινό Ευρωπαϊκό Συμβούλιο (IP/09/351) με σκοπό την αποκατάσταση της εμπιστοσύνης και την εξασφάλιση υπεύθυνων χρηματαγορών στο μέλλον. Η Επιτροπή έχει συνεργαστεί με την Επιτροπή Ρυθμιστικών Αρχών των Αγορών Κινητών Αξιών (ΕΡΑΑΚΑ) σχετικά με προτάσεις για ένα φιλικό προς τους επενδυτές ενημερωτικό δελτίο για προϊόντα των κεφαλαίων ΟΣΕΚΑ[1], τις «βασικές πληροφορίες για τους επενδυτές» (ΒΠΕ), που αποτελεί ένα καλό σημείο αναφοράς για άλλες δέσμες επενδυτικών προϊόντων για ιδιώτες επενδυτές. Οι κανόνες για τις πωλήσεις προϊόντων, είτε από μεσάζοντες είτε από προμηθευτές προϊόντων, καλύπτουν τομείς όπως οι συγκρούσεις συμφερόντων, τα κίνητρα και η καταλληλότητα ή η ορθότητα της πώλησης. Η Επιτροπή θεωρεί ότι οι σχετικές διατάξεις της οδηγίας για τις αγορές χρηματοπιστωτικών μέσων (MiFID) αποτελούν ικανοποιητικό μέτρο σύγκρισης για την πώληση όλων των δεσμών επενδυτικών προϊόντων για ιδιώτες επενδυτές.Περισσότερες πληροφορίες διατίθενται στην ιστοσελίδα: http://ec.europa.eu/internal_market/finservices-retail/investment_products_en.htm MEMO/09/210 [1] Oργανισμοί Συλλογικών Επενδύσεων σε Κινητές Αξίες

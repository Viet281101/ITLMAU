# Animation-2D-OpenGL-and-Gl4Dummies
2D GPU Animation using OpenGL and Gl4Dummies


## Description

This application uses OpenGL and GL4Dummies to make graphic models of fractals.  

Modeled fractals :

* Sierpinski triangle
* Mandelbrot set
* Julia set
* Pythagorean tree
* Barnsley fern

## Getting Started

### Dependencies

* OpenGL
* GL4Dummies

### Installing


### Executing program

* Run window.c
* Press "Q" or "ESCAPE" to close the window.
